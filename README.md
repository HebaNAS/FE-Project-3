Frogger - Arcade Game
===============================

###Starting the Game
Go to http://hebanas.github.io/FE-Project-3/


###How to play
Move player across the stone path to the water without hitting any bugs.

###Controls
- **Up** - up arrow
- **Down** - down arrow
- **Left** - left arrow
- **Right** - right arrow
