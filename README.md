Frogger - Arcade Game
===============================

###Starting the Game
Download the zip file and locate `index.html`.

Open `index.html` in a web browser.


###How to play
Move player across the stone path to the water without hitting any bugs.

###Controls
- **Up** - up arrow
- **Down** - down arrow
- **Left** - left arrow
- **Right** - right arrow
